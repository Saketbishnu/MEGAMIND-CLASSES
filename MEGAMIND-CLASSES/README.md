# MEGAMIND CLASSES

Production-ready MERN EdTech platform scaffold.

This repository is organized as a multi-workspace application:

- `frontend/` - student-facing React + Vite + Tailwind CSS application
- `admin/` - admin dashboard React + Vite + Tailwind CSS application
- `backend/` - Express + Node.js API for MongoDB Atlas, JWT auth, Cloudinary media, and platform services

Application code has intentionally not been added yet. This scaffold only defines the production folder architecture and placeholder files.
